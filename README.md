# SRE Monitoring Stack

Monitoring stack built with Docker, Prometheus, Grafana and Node Exporter.

## Architecture

Linux Host
   │
Node Exporter
   │
Prometheus
   │
Grafana

## Features

- Linux system metrics monitoring
- Prometheus metrics scraping
- Grafana dashboards
- CPU and memory alert rules

## Technologies

- Docker
- Prometheus
- Grafana
- Node Exporter
- Linux
